THIS REPO CONTAINS A BASH SCRIPT WHICH COLLECTS YOUR FILENAME AS A QUERY TO BE BLASTED AGAINST A DABASE WHICH YOU CAN INPUT (P.S: THE SCRIPT DOESNT CONTAIN THE MAKING OF THE PRTEIN HOMOLOG STUFF AS A DATABASE)
IT THEN TAKES YOUR OUTPUT, FILTER THE RESULTS SO THE PERCENTAGE IDENTITY IS GREATER THAN 90% AND SAVES IT AS YOUR QUERY NAME_FILTERED.TSV
THIS FILE IS THEN MATCHED AGAINST THE ARO_INDEX FILE GIVING AN OUTPUT THAT MATCHES THE ARO ID TO CORRESPONDING FAMILY AND RESISTANCE CLASS GIVING THE QUERY_RESISTANCE.TSV OUTPUT
